# Online Learning Platform

This is an online learning platform built using Next.js, React, Tailwind CSS, and PostgreSQL. The platform allows users to browse and learn from courses, track their progress, and create visually appealing courses using Google Generative AI.

## Features

- Browse and search for courses
- View course details, including chapters and progress tracking
- Enroll in courses and track progress
- Create courses with course banners and images generated by Google Generative AI
- User authentication and account management

## Tech Stack

- Frontend: Next.js, React, Tailwind CSS
- Backend: Next.js API routes, PostgreSQL, Prisma ORM
- Authentication: Clerk
- AI Integration: Google Generative AI

## Deployment

The project is deployed using Vercel, which provides a seamless deployment experience for Next.js applications.

## Getting Started

To get started with the project, follow these steps:

1. Clone the repository
2. Install dependencies: `npm install`
3. Create a `.env.local` file with your environment variables
4. Run the development server: `npm run dev`
5. Open your browser and visit `http://localhost:3000`


## Teck stack 
- Next.js
- React
- Tailwind CSS
- PostgreSQL
- Prisma ORM
- Clerk (for authentication)
- Google Generative AI (for course banner and image generation)
- Vercel (for deployment)
- TypeScript (for type safety and improved developer experience)

- React Query (for data fetching and state management)
- Axios (for making HTTP requests to the backend API)
